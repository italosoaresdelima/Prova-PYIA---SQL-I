# Prova-PYIA---SQL-I
Este projeto demonstra a criação e estruturação básica de uma tabela de banco de dados relacional utilizando SQL. O objetivo é aplicar conceitos fundamentais como tipos de dados, chaves primárias e propriedades de auto-incremento.   Tecnologias Utilizadas SQL (Structured Query Language)  Compatível com MySQL, PostgreSQL, SQLite e SQL Server.
